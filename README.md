# homebrew-tap

Homebrew tap for [Webstone](https://github.com/webstonehq) projects.

## Install

```sh
brew install webstonehq/tap/tuxedo
```

Or, equivalently:

```sh
brew tap webstonehq/tap
brew install tuxedo
```

## Formulae

- [`tuxedo`](Formula/tuxedo.rb) — fast, keyboard-driven terminal UI for [todo.txt](http://todotxt.org/). Source: [webstonehq/tuxedo](https://github.com/webstonehq/tuxedo).

## Updates

Formulae in this tap are updated automatically by the release pipeline of the
upstream project. The published formula always matches the latest GitHub
release.
